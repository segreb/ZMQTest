Проект сделан в Delphi XE 10.3 Community Edition.
Использована библиотека ZeroMQ версии 2.2.0. 
Исходники обёртки для Delphi взяты с https://github.com/bvarga/delphizmq/, с ветки master.
Лежат в каталоге ZMQ. 
Формат json, прежложенный в техзадании, не может быть обработан штатными библиотеками Delphi.
Использовалась библиотека superobject. Исходники лежат в каталоге superobject. 
Dll для платформ x86 и x64 помещены в каталоги к exe.
